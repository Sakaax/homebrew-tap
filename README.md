# sakaax/homebrew-tap

Tap Homebrew pour les apps [sakaax](https://sakaax.com).

## Cleared

Passe la review App Store du premier coup — analyse locale de tes builds iOS.

```sh
brew install --cask sakaax/tap/cleared
```

> Le binaire est hébergé sur `cleared.sakaax.com` (signé Developer ID + notarisé Apple).
> Ce dépôt ne contient que la formule d'installation, pas le code source de l'app.

Site : <https://cleared.sakaax.com>
